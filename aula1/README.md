# Automação de Cadastro de Produtos

Projeto desenvolvido em Python com o objetivo de automatizar o cadastro de produtos, simulando um processo comum em rotinas administrativas e sistemas internos.

## Objetivo
Automatizar o preenchimento e envio de dados de produtos, reduzindo tarefas manuais e erros humanos.

## Tecnologias utilizadas
- Python
- Biblioteca de automação - pyautogui
- Lógica de programação
